Car Dealership Database Management System
Описание проекта

Car Dealership Database Management System — это SQL-проект по управлению базой данных для автосалона. Проект решает задачи хранения и организации данных о клиентах, заказах, автомобилях и компаниях-производителях.

Особенности проекта:

Выполнен на SQL.

Поддерживает отслеживание заказов и клиентских данных.

Внешние ключи имплементируют связи между таблицами.

Комментарии для лучшего понимания структуры базы данных.
Проект выполнен с использованием JavaFX и PostgreSQL. База данных оптимизирована с применением внешних ключей для обеспечения целостности данных, а также с комментариями для удобства поддержки и масштабирования.
